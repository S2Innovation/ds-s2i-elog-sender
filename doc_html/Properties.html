MaxQueueSize Defines limit of number of entries pending. If it is reached the entries are not accepted and special substitution string ` {%MaxQueueMessage%}` is set on the last entry waiting in a queue. int 10
MaxQueueMessage Message to indicate that there may be missing entries. String There are {%NumberOfRejectedEntries%} entries skipped to avoid flooding.
ArgumentsParsers List of RegEx to parse an argument provided with the create_entry command.
Items of this property are applied to related items of the command DevVarStringArray argument (first item to the first item, second to second...).
If the regular expression contains named group (`(?P...)`) this group is provided dynamically to be used in other properites as replacement of `{%group_name}` string.
See also the AttributeFields property. 		String[] none
